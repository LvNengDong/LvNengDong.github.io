    什么是路由

    路由就是一组映射关系，即<K, V>键值对。

    在前端（Vue）中，K 是 Path，V 是 通过这个 Path 要访问的组件。

    Vue-Router 概述

    • Vue-Router 是官方提供的用来实现 SPA（Single Page Application） 的 vue 插件
    • GitHub地址：https://github.com/vuejs/vue-router
    • 官方文档（中文）：http://router.vuejs.org/zh-cn/
    • 下载：npm install vue-router --save

    通过上面两步，我们已经得到一个配置了路由信息的路由实例对象，但是这个实例对象是否可以直接使用呢？答案是否定的。虽然我们已经创建并配置了路由对象，但 Vue 是无法感知到的。

    所以，注册路由对象的意义就是将我们配置好的路由对象注册到项目的全局 Vue 对象中。对于我们配置好的 VueRouter 实例，如果不告诉 Vue 的话，Vue 在启动项目的时候是不会去加载 VueRouter 实例的。这有点类似于 SpringBoot 中的 @Component 注解，只有声明使用后项目才会去加载该类中的信息。

    Vue 项目一般是一个单页面应用，只会绑定一个全局 Vue 对象（当然这个 Vue 对象可以拥有多个子配置对象），我们需要在全局 Vue 实例中声明使用 VueRouter 实例，项目在加载时才会去加载 VueRouter 实例中配置的路由信息。

    • 路由组件：需要进行路由跳转的组件，一般放在 views/pages 目录下
