通过虚拟机安装CentOS

1 安装VMvare

1.1 什么是虚拟软件

虚拟原件是一个可以使你在一台机器上同时运行二个或更多Windows、LINUX等系统。它可以模拟一个标准PC环境。这个环境和真实的计算机一样，都有芯片组、CPU、内存、显卡、声卡、网卡、软驱、硬盘、光驱、串口、并口、USB控制器等。

1.2 常用的虚拟原件

  • VMware workstation
  • VirtualBox

1.3 VMware workstation安装

  1. 双击VMware-workstation-full-9.0.0-812388.exe

  2. 点击next进行安装

  3. 选择安装方式

  • Typical：典型安装
  • Custom：自定义安装

  1. 选择程序安装位置

    点击change选择程序安装位置，然后点击next

  2. 选择是否自动检测更新

    如果勾选，有新版时会提示你跟新版本，点next进行下一步

  3. 创建快捷方式，选择后点击next。

  4. 配置完成，开始安装程序

    点击continue。

  5. 开始安装虚拟原件

  6. 完成。

    点击finish完成安装。

注意：

虚拟软件安装完成后会在 控制面板\网络和Internet\网络连接 下多出来两个虚拟网卡VMware Network Adapter VMnet1和VMware Network Adapter VMnet8，需要将这两个网卡禁用，不然就无法收到视频了。

  • 选中VMnet1或VMnet8，右键，选中禁用

2 CentOS的安装

  • 环境：Windows7 , VMware Workstation10, CentOS6.4
  • 为什么选择CentOS ?
    1. 主流： 目前的Linux操作系统主要应用于生产环境，主流企业级Linux系统仍旧是RedHat或者CentOS
    2. 免费： RedHat 和CentOS差别不大，CentOS是一个基于Red Hat Linux 提供的可自由使用源代码的企业级Linux发行版本
    3. 更新方便：CentOS独有的yum命令支持在线升级，可以即时更新系统，不像RED HAT那样需要花钱购买支持服务！

2.1 安装步骤

  1. 首先安装VMware Workstation，只需点下一步即可，安装过程见1 安装Vmware。

  2. 文件 → 新建虚拟机 或 直接点击 创建新的虚拟机图标。

  3. 选择 典型（推荐）→ 下一步

  4. 稍后安装操作系统

  5. 选择操作系统和版本

  6. 输入虚拟机名称和安装路径

  7. 设置磁盘大小

  8. 自定义硬件

  9. 选择CentOS安装镜像文件

  10. 点击完成

  11. 启动虚拟机

  12. 选择第一项，安装全新操作系统或升级现有操作系统

  13. Tab键进行选择，选择Skip，退出检测

  14. 点击Next

  15. 选择语言，这里选择的是中文简体

  16. 选择键盘样式

  17. 选择存储设备

    如果以前安装过虚拟机，会出现这个警告，选择是，忽略所有数据

  18. 输入主机名

  19. 配置网络

  20. 设置时区，勾选使用UTC时间

  21. 输入根用户（root）的密码

    如果密码过于简单会出现提示，点击无论如何都使用

  23. 根据此Linux具体功能，选择不同的方式

  24. 选择现在自定义，自定义安装需要的软件，如桌面配置[这一步可以根据需求选择是否跳过]

    可以根据具体的情况来配置，如可安Eclipse

    还可以安装Java平台、Perl支持等

    选择语言支持

  25. 点击下一步，开始安装

  26. 安装完成后，点击重新导引

  27. 点击前进按钮

  28. 点击是，同意许可，再点击前进按钮

  29. 创建用户

  30. 设置日期和时间，如果可以上网，勾选在网上同步日期和时间

    最后点击前进，完成安装！

3、基本操作

  • 快捷键

  • 本机Linux账户密码

  • 本机中Linux系统的安装位置：D:\CentOS\Linux.vmx.lck

  • ifcofig：Linux中查看本机的IP地址。本机中Linux虚拟机的地址192.168.64.128

  • CRT窗口中的复制粘贴：选中内容后点击右键即可完成复制粘贴

VirtualBox安装CentOS7

1、VirtualBox安装CentOS7

  1. 下载安装VirtualBox。官网：https://www.virtualbox.org

  2. 打开VirtualBox –> 新建；

    因为Centos其实是RedHat的克隆版，所以版本类型选择RedHat（64-bit）

  3. 下一步，选择默认配置就好

  4. 下一步，

  5. 创建；

    在选择磁盘文件类型的时候主要考虑该Centos 7是否要被其他虚拟化软件(如vmware)使用

    • VDI：被VirtualBox使用
    • VHD：被Microsoft Virtual PC使用
    • VMDK：被Vmware使用

  6. 这里选择VDI类型，点击下一步

  7. 选择动态分配，下一步

  8. 点击创建，完成对虚拟机的创建

2、将CentOS7.iso文件与该虚拟机关联起来

  1. 点击设置 -> 存储–> 选择虚拟盘

  2. 系统–>主板，将光驱启动设置为第一项，点击 OK

  3. 点击启动，开始安装过程

  4. 选择Install Centos Linux 7

  5. 参考如下博客

https://www.cnblogs.com/alsodzy/p/9811175.html
