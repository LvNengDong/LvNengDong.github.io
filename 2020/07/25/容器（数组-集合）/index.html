容器（数组+集合）

1、容器

容器：是将多个数据存储到一起，每个数据称为该容器的元素。

2、数组(Array)

数组：数组是容器的一种，但是多了2个限制条件：

  1. 数组长度固定。
  2. 数组中元素的数据类型一致。

9.1 数组的定义(3种方式)

以方式1创建数组对象时，没有向数组中添加具体元素，但是JVM会自动将数组中所有元素赋值为0。

9.2 数组的索引

数组会自动为其中的每个元素分配一个**索引(index)**，索引从0开始。

9.2.1 索引的作用

  • 可以通过索引找到到数组中对应的元素。获取到元素后，可以：

    1. 获取元素；
    2. 给元素赋值。

1. 访问数组中的元素 & 为数组中的元素赋值

  • **数组名[索引]=数值**：为数组中的元素赋值
  • **数据类型 变量名 = 数组名[索引]**：取出数组中的元素。

2. 数组的长度属性

  • length是数组类中的一个属性。数组的长度是固定的，可以通过数组的长度属性length访问到数组的长度。
  • 数组的最大索引值为：length-1

9.3 数组在内存中的存储情况

  • 程序的执行流程：

    1. main()方法进入栈内存中执行。
    2. 执行new int[3]语句创建数组,JVM会在堆内存中开辟一块空间，存储数组。
      • 数组内容保存在堆内存中，而数组的地址则保存在栈内存中；
      • 数组中有3个元素，默认值是0.

9.4 数组中常见的操作

9.3.1 遍历

数组遍历：就是将数组中的每个元素分别或取出来。

数组遍历的方式

  1. for循环
  2. 增强for循环：for-each

数组不能使用迭代器Iterator遍历，因为Array类没有实现Iterable接口。*

9.5 数组中常见的异常

  1. 数组下标越界异常

    • ArrayIndexOutOfBoundException

  2. 数组空指针异常

    • NullPointerException

    数组空指针异常，意味着变量arr将不会保存堆内存中数组的地址，也就不允许再操作数组了。

9.6 数组的工具类：Arrays

java.util.Arrays：此类包含用来操作数组的各种方法。 该类中所有的方法均为静态方法，可通过类名直接调用

常用方法：

方法 说明
1 public static String toString（int[] a） 返回指定数组内容的字符串表示形式。
2 public static void sort（int[] a） 对int[] a数组进行升序排序

3、集合

3.1、Iterable接口

实现Iterable接口的对象可以使用for-each循环遍历，也可以使用iterator迭代器进行遍历。

除了实现此接口的对象外，数组也可以使用for-each循环遍历，但是数组不能使用iterator迭代器遍历。

Iterable接口的子接口：Collection接口

3.1.1、Iterable接口 vs. Iterator接口

接口源码：

Iterable接口中有一个抽象Iterator<T> iterator()，该方法的作用是返回一个Iterator实例，这个实例也被称为迭代器。

所有实现了Iterable接口的类必须重写iterator()方法，通过该方法可以获得一个迭代器，通过迭代器中的方法可以遍历集合。

3.2、Iterator接口

3.2.1 包

3.2.2 解释说明

  • 迭代：即Collection集合中元素的通用获取方式。
    1. 在取出元素之前要先判断集合中有没有元素，如果有，就把这个元素取出来，
    2. 继续判断下一个，如果有就再取出来。一直把集合中的所有元素全部取出来。

增强for循环

  • 增强for循环（也称for...each循环）是JDK1.5++推出的一个高级for循环，专门用来遍历集合和数组的。
  • 它的内部原理其实是一个Iterator迭代器，所以在遍历的过程中，不能对集合中的元素进行增删操作。

3.5 List接口

public interface List<E> extends Collection<E>

3.5.1. 解释说明

  • List接口继承自Collection接口，是单列集合的一个重要分支。我们一般把实现了List接口的对象称为List集合。
  • 在List集合中允许出现重复的元素，
  • 所有的元素是以一种线性的方式进行存储的，在程序中可以通过索引来访问集合中的指定元素。
  • List集合元素有序，即元素的存入顺序和取出顺序一致。

List接口的特点

  1. 元素有序（存入和取出顺序一致）
  2. 元素可重复
  3. 带有索引，通过索引就可以操作集合中的元素。
